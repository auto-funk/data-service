Data Generator

What is DataGenerator?

DataGenerator provides you to create randomly entities. You can add models which will be saved and reused to create data.

What can it generate?

DataGenerator is able to generate random data for all the model already saved.

Add model to database

To create entities, you have to write a json request like this :

    {
        "model": {
            "properties": [
                {
                    "name": "firstName",
                    "type": "string"
                },
                {
                    "name": "lastName",
                    "type": "string",
                    "description": "Nom de la personne"
                },
                {
                    "name": "age",
                    "type": "integer"
                }
            ],
            "metadata": {
                "name": "People",
                "description": "Des personnes"
            }
        }
    }

Generate entities

To generate entities, you need a model already saved in the database. You need to add the model name in the URL like this for the author model: 

    http://33.33.33.10.xip.io/authors

The application will provide you a JSON or XML object.

    {
        "firstName":"Jacques",
        "lastName":"Lefevre",
        "email":"jacques.lefevre@gmail.com",
        "birthDate":"09-06-1988"
    }
